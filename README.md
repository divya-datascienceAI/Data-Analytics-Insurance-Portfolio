# 🚗 Auto Insurance Claims Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes auto insurance policies and claims data. The dashboard provides insights into customer profiles, policy coverage, claim trends, fraud patterns, and financial performance to support data-driven decision-making for insurance companies.

---

## 🎯 Business Objective

The objective of this project is to:

- Analyze insurance claim trends and customer behavior.
- Identify factors contributing to high claim amounts.
- Monitor fraud rates across different customer segments.
- Evaluate policy coverage and premium performance.
- Support underwriting and risk management decisions.

---

## 🛠️ Tools & Technologies

- **Power BI** – Data Visualization
- **Power Query** – Data Cleaning & Transformation
- **DAX** – Measures & Calculations
- **Excel** – Data Source

---

## 📊 Dashboard Highlights

The interactive dashboard includes:
- Executive KPI Summary
- Total Policies
- Total Claim Amount
- Average Annual Premium
- Average Claim Amount
- Fraud Rate (%)
- Customer Demographics Analysis
- Policy Coverage Analysis
- Incident & Claim Analysis
- Auto Make vs Claim Type Analysis
- Number of Vehicles Involved vs Claim Amount
- Total Claim Amount by Incident Type
- Fraud Rate by Education Level
- Umbrella Coverage Analysis
- Premium vs Claim Analysis
- Interactive slicers for dynamic filtering
